Hibernate - QCM - Quiz des formations Hibernate
===============================================

Pour faire jouer les stagiaires :
* leur fournir le package des tps et leur faire ouvrir la page html en local sur leur poste
* démarrer le mini webserveur sur le poste du formateur (java -jar SimpleWebServerFull.jar) et les faire se connecter depuis leur poste de formation


Pour enrichir un questionnaire :
* modifier un des fichiers qcm*.js


Pour ajouter un questionnaire :
* ajouter un fichier qcm*.js dans le même répertoire que les autres questionnaires déjà existant (documentation ici : https://github.com/jewlofthelotus/SlickQuiz)
* ajouter une ligne après la ligne 34 dans le fichier index.html référençant le nouveau quiz
